What is scope

It is an attribute added to the (th) element in tables to specify whether the header applies to a row or a column. For example,scope="col" means it's a column header, and scope="row" means it's a row header.

how does it work to support accessibility:

When we define the scope, tools like screen readers can understand the relationship between headers and data in the table, making it easier for people who can't clearly see the table to understand the content.
